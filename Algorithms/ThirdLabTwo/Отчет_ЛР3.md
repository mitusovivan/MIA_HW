## 1. Цель работы
Исследовать эффективность базового BFS и его модификаций для задачи поиска всех вершин дерева, находящихся на заданном расстоянии `Z` от целевой вершины.

## 2. Описание задачи
Дано двоичное дерево (генерируется в программе).  
Необходимо найти все вершины, расстояние от которых до целевой вершины равно `Z`.

Сравниваются алгоритмы:
1. Базовый BFS от целевой вершины.
2. Двунаправленный BFS.
3. Подход с предподсчётом LCA.
4. Подход с двоичными подъёмами (Binary Lifting).

## 3. Теоретическая часть

### 3.1. Базовый BFS
Поиск в ширину от целевой вершины выполняется по уровням.  
Когда глубина обхода достигает `Z`, вершина добавляется в ответ.

### 3.2. Двунаправленный BFS
Расстояние между парой вершин вычисляется двумя фронтами поиска (из `u` и `v`).  
Это может сократить число просмотренных вершин на больших графах.

### 3.3. LCA с предподсчётом
После вычисления `parent[]` и `depth[]`:
\[
dist(u,v)=depth[u]+depth[v]-2\cdot depth[lca(u,v)]
\]
Для каждой вершины проверяется равенство `dist(target, node) == Z`.

### 3.4. Binary Lifting
Используется таблица предков `up[k][v]` (предок вершины `v` на высоте `2^k`).  
LCA вычисляется быстрее за счёт подъёмов по степеням двойки.

## 4. Формализация и спецификация
- Вход: дерево `adjacency`, целевая вершина `target`, расстояние `Z`.
- Выход: отсортированный список вершин, удалённых от `target` ровно на `Z`.
- Требования интерфейса:
  - генерация дерева по `N` и `seed`;
  - выбор целевой вершины числом или кликом на визуализации;
  - запуск выбранных алгоритмов;
  - таблица сравнения времени, операций и результата.

## 6. Контрольный пример
1. Сгенерировать дерево `N = 50` при фиксированном `seed`.
2. Выбрать `target = 7`, `Z = 5`.
3. Запустить базовый и модифицированные алгоритмы.
4. Проверить совпадение множеств найденных вершин.

## 7. Анализ результатов
- Все алгоритмы должны давать одинаковый набор вершин.
- Отличия ожидаются по времени и числу операций.
- Для единичного запроса базовый BFS обычно проще и эффективен.
- При множественных запросах выгодны предподсчёты (`LCA`, `Binary Lifting`).

## 8. Улучшение алгоритма
Выполнены модификации относительно базового BFS:
- двунаправленный поиск;
- LCA с предподсчётом;
- двоичные подъёмы.

Также добавлена визуальная верификация: подсветка найденных вершин и сравнение результатов в таблице.

## 9. Вывод
Проведено сравнение нескольких подходов для задачи поиска вершин на расстоянии `Z` в дереве.  
Базовый BFS остаётся удобным эталоном, а методы с предподсчётом дают преимущества в сценариях повторных запросов.
